Installation
============

The extension needs to be installed via Composer as any other extension of TYPO3 CMS in a Composer based installation.
Because of dependencies with other libraries it is not possible to use it without Composer.

#. Get the extension
   #. **Usage with composer**: Use `composer require ehaerer/easy-verein`.


Latest version from git
-----------------------
You can get the latest version from git by using the git command:

.. code-block:: bash

   git clone https://github.com/Kephson/easy_verein.git

.. important::

   The main branch supports TYPO3 11.5 only.
